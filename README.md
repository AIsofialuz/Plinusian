# Plinusian

A comprehensive smart contract security audit tool leveraging **Fable 5** to identify and validate vulnerabilities through proof-of-concept (PoC) exploits.

## Overview

Plinusian is an advisory-focused security scanner designed to audit Fable 5 smart contracts for vulnerabilities. Rather than relying on assumptions, Plinusian confirms every finding by generating executable PoCs that demonstrate the actual exploitation of discovered issues.

**Key Focus:** Non-owner exploitable vulnerabilities across 40 classes of DeFi vulnerabilities

## Features

- **Comprehensive Vulnerability Scanning** - Analyzes smart contracts across 40 established DeFi vulnerability classes
- **No-Skip Analysis** - Ensures exhaustive coverage with no vulnerabilities overlooked
- **Proof-of-Concept Generation** - Automatic PoC creation to validate each discovered vulnerability
- **Non-Owner Exploitation Focus** - Prioritizes vulnerabilities exploitable by non-privileged actors
- **Advisory-Grade Reporting** - Detailed, objective findings without assumptions or vague language
- **Rigorous Validation** - Every finding is confirmed through executable PoCs before reporting
